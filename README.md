# Corona-detection-project

Project title: corona detection and management system 

overview: this a complete desktop application developed in python that facilitates covid-19 risk assessment and patient management. the system allows healthcare provides to register patents, conduct corona symptom evaluations, automatically calaculate risk scores and store test result in a secure database

key features:
patient registration and profile management 
symptom-based corona risk assessment
automated test result classification(positive/negative)
severity level dtermination (low/medium/high)
analytical dashboard with visual charts
secure SQLite databse storage
user friendly Tkinter GUI interface 

Techinal Stack:
frontend: Tkinter (python)
Backend:: Python
Database: SQLite
Visualization: Matplotlib
Impact: Help healthcare facilities efficiently manage COVID-19 screening, maintain organized patient records and generate valuable insights through data analytics
